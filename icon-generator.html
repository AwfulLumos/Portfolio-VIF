🔧 How to Generate Missing Icons:

Option 1: Online Converter (Easiest)

  1. Go to favicon.io/favicon-converter
  2. Upload icon.svg
  3. Download the package
  4. Copy files to your project root:
    • favicon.ico → root directory
    • apple-touch-icon.png → root directory
    • android-chrome-192x192.png → rename to assets/images/icon-192.png
    • android-chrome-512x512.png → rename to assets/images/icon-512.png

Option 2: Using ImageMagick (If Installed)

Run these commands in PowerShell from your project root:

magick icon.svg -resize 192x192 assets/images/icon-192.png
magick icon.svg -resize 512x512 assets/images/icon-512.png
magick icon.svg -resize 180x180 apple-touch-icon.png
magick icon.svg -resize 32x32 favicon.ico
